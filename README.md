# A4P Platform V7 Final

Cette version est conçue pour être visible immédiatement sur GitHub Pages.

## Déploiement

1. Crée un dépôt GitHub neuf.
2. Uploade **le contenu de ce dossier** à la racine du dépôt.
3. Vérifie que `index.html` est bien à la racine.
4. Active GitHub Pages : Settings > Pages > Deploy from branch > main > /(root).

## Résultat attendu

L'URL du dépôt doit afficher immédiatement la page d'accueil complète de la plateforme.
